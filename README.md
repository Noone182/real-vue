# real-vue

## Project setup
```
npm install
```
### Before run serve, install and run a json-server
```
npm i -g json-server
json-server --match db.json
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
